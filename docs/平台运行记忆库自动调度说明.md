# 平台运行记忆库自动调度说明

## 1. 目标

将“每日操作日报 -> 平台运行记忆库”做成自动化任务，服务启动后自动检查，并按计划时间完成日报入库。

## 2. 默认规则

- 计划时间：每天 `00:10`
- 入库对象：前一天的操作日报
- 自动任务：服务启动后后台线程持续检查
- 补跑方式：支持手动触发指定日期或触发待补日期

## 3. 状态查看

可以通过以下接口查看自动任务状态：

- `GET /operations/daily-report/automation`

状态包括：

- 是否已启动
- 计划时间
- 上次成功日期
- 下次待跑日期
- 最近错误

## 4. 手动补跑

可以通过以下接口手动补跑：

- `POST /operations/daily-report/automation/run`

如果不传日期，会自动补跑当前待处理的日报；如果传 `date=YYYY-MM-DD`，则只跑指定日期。

## 5. 首页入口

知识平台首页已增加“日报自动任务”卡片，可直接查看状态并手动补跑。

## 6. 后续扩展

后续可以把调度时间改成可配置项，并把自动任务状态写入平台运行总览页。
