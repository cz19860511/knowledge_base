# 知识库目录说明

本目录按新版实施计划重建，当前聚焦 `前期资料预处理 -> 生成 chunk -> 生成向量库 -> RAG 检索`。

## 目录用途

- `raw/`：原始资料，只保留源文件，不做修改。
- `staging/`：预处理临时区，主要用于把 `02-07` 目录串成 MinerU 可处理的输入。
- `working/parsed/`：文档预处理产物，按批次保留 MinerU 和 MarkItDown 输出。
- `working/evaluation/`：本轮预处理评估汇总、候选清单和复核结果。
- `selected/`：人工确认后的主版本知识文档，当前以 `batch_20260521/documents/` 形式保存每份 `selected.md`。
- `chunks/`：后续切片产物，当前输出 `chunks.jsonl`、`chunk_stats.json`、`chunks_preview.csv` 和 `chunks_preview.xlsx`。
- `vectors/`：后续向量库产物。
- `rag/`：后续 RAG 检索测试与结果记录。
- `packages/`：对 Agent 或平台输出的交付包。
- `operations/`：变更、问题和反馈记录。
- `metadata/`：知识库元数据和口径定义。
- `inventory/`：资料盘点、版本核查和重复核查清单。
- `qa/`：问答测试集与回归结果。
- `流程配置说明文档.md`：MinerU、Chunk、Embedding 流程配置项说明。
- `参数推荐值与调参建议.md`：流程配置的短版推荐值和调参速查。
- `多知识库管理方案.md`：多知识库平台化的目录、接口和实现方案。
- `知识平台闭环与自进化方案.md`：资产版本、操作日志、日报入库和自进化闭环方案。
- `资产版本链设计.md`：原始文件、中间产物、向量和日报资产的版本链统一设计。
- `版本对账机制.md`：操作事件和资产版本链的对账逻辑与输出说明。
- `平台事件回放机制.md`：任意一天的平台事件时间线、日报和对账回放说明。
- `平台总览页说明.md`：平台总览页的指标、入口和使用方式说明。
- `平台任务页说明.md`：平台任务列表、详情、流转、日志、提醒、周报和自动汇总说明。
- `平台任务自动汇总调度说明.md`：任务到期提醒和周报自动调度说明。
- `大模型接入知识平台方案.md`：大模型接入的三层结构、接口边界和平台衔接方案。
- `大模型接入接口清单与数据流图.md`：大模型接入的接口清单、请求流和数据流图。
- `大模型接入实施任务表.md`：大模型接入知识平台的实施任务拆解和里程碑。
- `知识平台检索接口契约.md`：`/knowledge-bases/retrieve` 的请求、返回和路由契约。
- `知识库路由规则说明.md`：多知识库场景下的路由优先级和冲突处理说明。
- `AgentArts联调说明.md`：AgentArts `General` 第三方知识库联调步骤和判定标准。
- `AgentArts联调请求包.md`：可直接复制的联调请求、冲突验证和成功判定样例。
- `知识平台操作事件规范.md`：知识平台关键操作事件的字段、范围和存储规范。
- `知识平台日报规范.md`：平台每日日报的输出位置、章节结构和生成原则。
- `平台运行记忆库入库说明.md`：日报入库到平台运行记忆库的目标目录、结果和触发方式。
- `平台运行记忆库自动调度说明.md`：日报自动入库的计划时间、状态查询和手动补跑说明。
- `知识平台自进化建议规范.md`：平台运行事件转成结构化建议时的字段和分类规范。
- `知识平台自进化建议模板集.md`：参数优化、回滚、补资料三类建议的标准模板。
- `自进化建议人工确认流程.md`：建议确认的决策类型、记录字段和使用原则。
- `平台任务闭环机制.md`：建议转任务、任务台账和任务状态闭环说明。
- `平台状态总结提示词.md`：大模型读取日报和建议后输出状态总结的提示词模板。
