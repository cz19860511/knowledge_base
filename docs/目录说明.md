# 知识库目录说明

本目录按新版实施计划重建，当前聚焦 `前期资料预处理 -> 生成 chunk -> 生成向量库 -> RAG 检索`。

## 目录用途

- `raw/`：原始资料，只保留源文件，不做修改。
- `staging/`：预处理临时区，主要用于把 `02-07` 目录串成 MinerU 可处理的输入。
- `working/parsed/`：文档预处理产物，按批次保留 MinerU 和 MarkItDown 输出。
- `working/evaluation/`：本轮预处理评估汇总、候选清单和复核结果。
- `selected/`：人工确认后的主版本知识文档，当前以 `batch_20260521/documents/` 形式保存每份 `selected.md`。
- `chunks/`：后续切片产物，当前输出 `chunks.jsonl`、`chunk_stats.json`、`chunks_preview.csv` 和 `chunks_preview.xlsx`。
- `vectors/`：后续向量库产物。
- `rag/`：后续 RAG 检索测试与结果记录。
- `packages/`：对 Agent 或平台输出的交付包。
- `operations/`：变更、问题和反馈记录。
- `metadata/`：知识库元数据和口径定义。
- `inventory/`：资料盘点、版本核查和重复核查清单。
- `qa/`：问答测试集与回归结果。
- `流程配置说明文档.md`：MinerU、Chunk、Embedding 流程配置项说明。
- `参数推荐值与调参建议.md`：流程配置的短版推荐值和调参速查。
